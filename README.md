# Minesweeper

A polished single-file HTML Minesweeper game built with Material Design 3, with no external dependencies.

## Live Demo

GitHub Pages: https://cizuwuxin.github.io/minesweeper/

## Features

- Material Design 3 visuals
- Classic and no-guess modes
- Custom difficulty settings
- Theme toggle and question marks
- Sound effects
- First-click safety
- Undo and restart
- Responsive layout for desktop and mobile

## Controls

| Action | Mouse | Touch | Keyboard |
|---|---|---|---|
| Reveal | Left click | Tap | Arrow keys + Enter |
| Flag | Right click | Long press | F |
| Chord | Double-click | — | Double-click |
| Undo | Toolbar | Toolbar | Ctrl+Z |
| New game | Toolbar | Toolbar | R |

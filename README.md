# Minesweeper

![Screenshot](./screenshot.png)

A single-file HTML Minesweeper game using Material Design 3, with no external dependencies.

🌐 [English](./docs/README.en.md) · [简体中文](./docs/README.zh-CN.md)

## Play

👉 https://cizuwuxin.github.io/minesweeper/

Works on phones, tablets, and desktops. Follows system light/dark theme, with a manual toggle in the game.

## Controls

| Action | Mouse | Touch | Keyboard |
|---|---|---|---|
| Reveal | Left | Tap | Arrow keys + Enter |
| Flag | Right | Long press | F |
| Chord | Double-click | — | Double-click |
| Undo | Toolbar | Toolbar | Ctrl+Z |
| New game | Toolbar | Toolbar | R |

## Features

- Material Design 3 visuals
- Classic and no-guess dual modes
- Custom difficulty (rows / cols / mines)
- Theme, question marks, sound effects
- First-click safety
- Undo and restart
- Responsive layout, mobile-friendly

## Deployment

Just open `index.html` locally, or deploy to any static host (e.g. GitHub Pages).

## 特点

- 经典 / 无猜双模式
- 单文件，零依赖，下载即玩
- 适配桌面与触控

## 能力

- 棋盘：简单 / 中等 / 困难 / 自定义 / 随机
- 辅助：提示、撤销、问号标记、进度条
- 设置：深色模式、音效、左右键互换、长按延迟

## 操作

| 目标 | 操作 |
| :-- | :-- |
| 打开格子 | 点击 / 左键 / `Enter` / `Space` |
| 标记格子 | 长按 / 右键 / `F` |
| 提示 | `H` |
| 撤销 | `Ctrl + Z` |
| 重开 | `R` |

<details>
<summary>本地打开</summary>

下载 `index.html` 直接用浏览器打开。或：

```bash
git clone https://github.com/cizuwuxin/minesweeper.git
cd minesweeper
python3 -m http.server 8000
```

访问 `http://localhost:8000`

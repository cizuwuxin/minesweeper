# 扫雷

一个用 Material Design 3 重写的扫雷游戏。整个项目就一个 HTML 文件，零依赖，双击就能玩。

**在线试玩：** https://cizuwuxin.github.io/saolei/saolei.html

![游戏截图](preview.jpg)

## 怎么玩

直接点上面链接，浏览器打开就完了。手机、平板、电脑都行，会自动跟系统切换深色 / 浅色主题。

## 操作

| 操作 | 鼠标 | 触屏 | 键盘 |
|---|---|---|---|
| 翻开格子 | 左键 | 轻点 | 方向键 + Enter |
| 标记雷 | 右键 | 长按 | F 键 |
| 数字展开 | 双击 | 双击 | 双击 |
| 撤销 | 顶栏 | 顶栏 | Ctrl+Z |

## 难度

| 难度 | 尺寸 | 雷数 |
|---|---|---|
| 简单 | 9 × 9 | 10 |
| 中等 | 16 × 16 | 40 |
| 困难 | 16 × 30 | 99 |
| 自定义 | 随便 | 随便 |

## 几个值得一提的实现

- **无猜模式** — 内置 CSP（约束满足问题）求解器，用变量消元和组件分析自动识别能确定的格子，不用瞎猜。
- **并行求解** — 求解器在 Web Worker 池里跑，根据 CPU 核数自动调线程，主线程不卡。
- **首次点击保护** — 第一下永远是安全的。
- **撤销** — 最多 50 步，用 Uint8Array 扁平快照，O(1) 还原。
- **Material 3** — 完整用 `--md-sys-color-*` token 体系，主题色改一个变量就能换。
- **响应式** — 360px 到 4K 显示器都能玩。

## 本地跑

```bash
git clone https://github.com/cizuwuxin/saolei.git
# 浏览器打开 saolei.html 就行
```

## 自己改

浏览器 DevTools 改 CSS 变量换主题色：

```css
:root {
  --md-sys-color-primary: #6750A4;  /* 改成你喜欢的颜色 */
}
```

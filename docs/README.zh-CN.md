# 扫雷

![游戏截图](./screenshot.png)

单文件 HTML 扫雷游戏，使用 Material Design 3 视觉风格，无外部依赖。

🌐 [English](./README.en.md) · [简体中文](./README.zh-CN.md)

## 试玩

👉 https://cizuwuxin.github.io/minesweeper/

手机、平板、电脑都能玩，自动跟随系统的浅色/深色主题，也可以在游戏内手动切换。

## 操作

| 操作 | 鼠标 | 触屏 | 键盘 |
|---|---|---|---|
| 翻开格子 | 左键 | 轻点 | 方向键 + Enter |
| 标记雷 | 右键 | 长按 | F |
| 数字展开 | 双击 | — | 双击 |
| 撤销 | 顶栏 | 顶栏 | Ctrl+Z |
| 新游戏 | 顶栏 | 顶栏 | R |

## 特点

- Material Design 3 视觉系统
- 传统 / 无猜双模式
- 自定义难度（行列/雷数）
- 主题、问号标记、音效反馈都可调
- 首次点击保护
- 撤销 / 重开
- 响应式布局，手机友好

## 部署

直接双击 `index.html` 即可在本地运行。也可部署到任何静态托管（如 GitHub Pages）。

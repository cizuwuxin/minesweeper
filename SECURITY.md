# 安全策略

## 支持的版本

| 版本 | 支持状态 |
|------|---------|
| main 分支最新版 | ✅ 支持 |
| 旧版 | ❌ 不再支持 |

## 报告漏洞

如发现安全漏洞，请**不要**通过公开 Issue 提交，而是：

1. 通过 GitHub 的 [Private vulnerability reporting](https://github.com/cizuwuxin/saolei/security/advisories/new) 提交
2. 或发送邮件到 cizuwuxin（GitHub 用户名对应邮箱）

请包含：
- 漏洞描述与影响范围
- 复现步骤
- 涉及的浏览器/版本
- 你的联系方式（可选）

## 响应承诺

- 48 小时内确认收到
- 7 天内评估严重程度并给出修复计划
- 修复后会在 Release Notes 中致谢（除非你希望匿名）

## 说明

这是一个纯前端单文件 HTML 项目，无后端、无数据库、无用户数据收集，安全风险面较小。但仍会认真对待每一个报告。
